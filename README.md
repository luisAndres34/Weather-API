# Temperature Checker

A lightweight, interactive command-line tool written in Python that fetches and displays the real-time temperature of any given city. This script uses the [wttr.in](https://wttr.in) weather API to retrieve accurate, up-to-date weather data.

## Features
* **Real-time Data:** Fetches the current temperature in Celsius.
* **Continuous Loop:** Check the weather for multiple cities without needing to restart the program.
* **Error Handling:** Notifies the user if a city is not found or typed incorrectly.

## Prerequisites
To run this script, you will need **Python 3** and the `requests` library installed. 

You can install the required library using pip:
```bash
pip install requests
```

## Usage
1. Download or clone the script to your local machine.
2. Open your terminal or command prompt.
3. Navigate to the folder containing the script and run:
```bash
python main.py
```

4. Follow the on-screen prompts to enter a city name, or type `e` to exit the program.

## Example Output
```text
Temperature checker!
Type your city to check its temperature or type 'e' to exit the program:
London
Currently the temperature in your city is 14°C
```
